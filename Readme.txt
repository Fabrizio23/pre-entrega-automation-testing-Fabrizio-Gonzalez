Para correr este proyecto es necesario instalar Python agregandolo al path (add to path) durante la instalacion
https://www.python.org/downloads/ (descarga Python)

Instalar Selenium
pip install selenium pytest (desde CMD)

Instalar el webdriver de Chrome
pip install chromedriver-autoinstaller (desde CMD)

Para ejecutar las pruebas se necesita abrir la terminal y ejecutar el archivo run_test.py
python run_tests.py
